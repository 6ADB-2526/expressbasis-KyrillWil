# Bouw een server op die aan onderstaande zaken voldoet.

Als basis krijg je een array met winkel items.
Er zit een gitignore file in --> LAAT DIT STAAN!

## wat moet er mogelijk zijn? 

Alles op vlak van return gebeurt via een JSON bestand

1. Een basis route met een 'welkom' als return
2. Een route: items --> als return krijg je alle items (enkel naam)
3. Een route: items/ID --> als je een id mee geeft krijg je de details te zien van dit item.
4. Een route: deleteItem/ID --> je verwijdert 1 item uit de lijst. Als return krijg je alle overgebleven items tezien. 
5. Een route: addItem --> via post/JSON voeg je een item toe aan het einde van de lijst. 
(je voegt 'snoep' toe -> als omschrijving: de lekkerste fruitige snoepjes)

Je mag gebruik maken van je notities en voorgaande oefeningen om deze toets te maken. 
Je maakt geen gebruik van AI.
